# Habit_Tracker

You can visit online version on https://tracker-of-habits.netlify.app/

This Single Page web application aims to provide a simple and easy-to-use interface for tracking and managing daily habits. It will allow users to add, remove and edit habits, as well as view their progress and streaks over time. The application will also provide a way for users to categorize their habits as either positive or negative, and to view their habits grouped by type. The goal of this application is to help users develop and maintain healthy habits, help the said user to get rid of their negative habit, and to provide a useful tool for tracking and managing their daily routines.
